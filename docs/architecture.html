Catalyst Components — System Architecture

Factory Inventory Management System · Vue 3 SPA + FastAPI backend + in-memory JSON data (no database)

System Overview

Frontend — Vue 3 SPA

localhost:3000 · Vite
  • 7 views: Dashboard, Inventory, Orders, Spending, Demand, Backlog, Reports
  • App.vue — nav, global styles, view switching
  • FilterBar.vue — 4 global filters
  • Composables: useFilters, useI18n (EN/JA), useAuth
  • Custom SVG charts, CSS Grid layouts
⇄HTTP
axios
api.js

Backend — FastAPI

localhost:8001 · uvicorn
  • main.py — 15 GET endpoints under /api/*
  • Query-param filtering: warehouse, category, status, month
  • Pydantic response models validate every payload
  • CORS enabled for the Vite dev origin
  • Swagger UI at /docs
→loaded at
startup
mock_data.py

Data — JSON files

server/data/ · in-memory
  • inventory.json
  • orders.json · purchase_orders.json
  • demand_forecasts.json
  • backlog_items.json
  • spending.json · transactions.json
  • Regenerate with generate_data.py

Request / Data Flow

User changes a filter→ useFilters reactive state→ api.js builds query params→ FastAPI endpoint filters in memory→ Pydantic validates response→ View refs update→ Computed properties re-render charts
